# Titanic Survival Prediction

End-to-end machine learning classification project based on the Kaggle Titanic dataset.

## Overview
This project aims to predict passenger survival on the Titanic using machine learning techniques. The workflow includes data preprocessing, feature encoding, model training, and evaluation.

## Highlights
- Data cleaning and preprocessing
- Feature encoding
- Random Forest modeling with hyperparameter tuning
- Achieved 0.779 accuracy on Kaggle leaderboard

## Tools
Python, Pandas, NumPy, Scikit-learn
